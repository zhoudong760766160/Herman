chapter1 ----Thread
线程与进程：
    1.线程是指程序执行过程中，能够执行程序代码的一个执行单元
    2.进程是指一段正在执行的程序
    3.线程是进程的实际运作单位，一个进程可以有多个线程

线程的生命周期：
    1.创建 new
    2.就绪 start()
    3.运行 run()
    4.阻塞
    5.死亡

使用多线程的原因：
    1.减少程序响应时间，增强程序交互效果友好性
    2.多核心CPU本身就具备多线程计算能力，使用单线程浪费资源
    3.使用多线程能简化程序结构
